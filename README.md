# 📋 产品需求看板

一个简洁的产品需求管理看板，支持 Vercel 一键部署。

## 预览

看板展示三个阶段的需求：
- 🔥 **NOW** - 进行中
- 📋 **NEXT** - 待排期
- 💡 **LATER** - 规划中

点击卡片可查看详情，包括范围、目标、里程碑和待办事项。

## 部署

### Vercel 部署

1. Fork 这个仓库
2. 在 [Vercel](https://vercel.com) 中导入项目
3. 自动部署完成 ✨

### 本地运行

```bash
npx serve .
```

## 更新数据

编辑 `data/roadmap.json` 即可更新看板内容。

## 技术栈

- 纯静态 HTML/CSS/JS
- 无依赖，无构建步骤
- 深色主题 UI
